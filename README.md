# workmotion
Cypress, Javascript, mochawesome report

# Pre Condition
Install the below packages
  1. Cypress
  2. Mocha
  3. Mochawesome
  4. Mochawesome-merge
  
# Steps to Run:
## Method 1: 
    Run the "runner.bat" file

## Method 2: 
    Open the project in vscode
    Run npm install
    Run command - node_modules\.bin\cypress run --headed --spec "cypress\integration\TestCases\TC_001.js" in vscode terminal
